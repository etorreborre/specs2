Companion object Env

case class Env(arguments: Arguments = Arguments(), selectorInstance: (Arguments) ⇒ Selector = ..., executorInstance: (Arguments) ⇒ Executor = ..., lineLogger: LineLogger = NoLineLogger, statsRepository: (Arguments) ⇒ StatisticsRepository = ..., systemLogger: Logger = consoleLogging, random: Random = new scala.util.Random, fileSystem: FileSystem = FileSystem, executionParameters: ExecutionParameters = ExecutionParameters()) extends Product with Serializable

Whole creation / execution / reporting environment for a specification

Part of it is driven by the command-line, part of it is specs2 implementation Yet it can be used to drive the creation or execution of examples

Note: creating an Env instance is a delicate activity because the Env hold a thread pool that needs to be shutdown at the end of the execution
